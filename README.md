# Hilios Clone

A modern clone of the Hilios project with enhanced features and improved UI/UX.

## Features

- Responsive design
- Modern UI components
- Enhanced user experience
- Performance optimized

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Technologies

- React.js
- Next.js
- Tailwind CSS
- TypeScript
